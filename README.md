# Expense Tracker

A simple **Expense Tracker** built with **Java and Spring Boot** that demonstrates building REST APIs and performing CRUD operations to manage financial records.

---

## 🚀 Overview

The application allows users to track and manage expenses through RESTful endpoints.
Users can add, view, update, and delete expense records stored in a relational database.

---

## 🛠️ Tech Stack

### Backend

* Java
* Spring Boot
* Spring Data JPA
* REST APIs

### Database

* MySQL

---

## ✨ Features

* Add new expenses
* View all expenses
* Update existing expense records
* Delete expenses

## ⚙️ Running the Project

### Clone the repository

```
git clone https://github.com/kayanr/ExpenseTracker.git
```

### Create database

```
CREATE DATABASE expense_tracker_db;
```

### Run the application

```
mvn spring-boot:run
```

Application runs at:

```
http://localhost:8080
```
